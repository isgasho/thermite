fn select_negative(self, neg: Self, pos: Self) -> Self

On platforms with true "select" instructions, they often only check the HSB, which happens to correspond to the "sign" bit for both floats and twos-compliment integers, so we can save a cmpgt(self, zero) by calling this

On platforms without true "select" instructions, this falls back to self.is_negative().select(neg, pos)
